## Easylancer Data API

The representational data api for the platform built with [Nest](https://github.com/nestjs/nest)

## Installation

```bash
$ npm install
```

## Running the app

The config folder is located in `./config`

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
